# SavrBites project rules

- Design comes before app logic.
- Read `docs/FIGMA_RULES.md` before building UI.
- Do not use a mono `App.tsx` or `App.jsx`.
- Keep `App.tsx` as a tiny entry point.
- Build under `src/` with feature screens, shared components, hooks, theme files, and types.
- Target mobile and tablet only: iPhone, iPad, Android phones, and Android tablets.
